Developer API

Advanced guide for extending Mint via Python and custom logic gates.

Python Integration

py(code)

Executes raw Python strings. Semicolons separate multiple statements.

py(print("Run Python"); x = 10; print(x))

Custom Command Templates

Standard Command

Basic structure for new tags in mint.py.

elif tag == "command":
    try:
        print("Success")
    except Exception as e:
        print(f"Error: {e}")
        return

Multiline Logic

Handles codes split by ; inside a single tag.

elif tag == "command":
    parts = val.split(",", 1) 
    commands = parts[1].split(";")
    for cmd in commands:
        # Run sub-command logic
        pass

Module Management

Edit mod.bat at line 13-15 to include new libraries.

pip install library_name

Remove pause to hide the console after setup.

Contributor Workflow

Follow these steps to sync core changes with the VS Code extension:

  • Modify syntaxes/mint.tmLanguage.json for highlighting.
  • Update package.json or extension.js if needed.
  • Finally run vsce package in the terminal to build.
