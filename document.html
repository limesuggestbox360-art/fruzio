Command Dictionary

A formal breakdown of the Mint language instruction set, parameters, and expected behaviors.

1. Basic Commands

write(string)

Outputs literal text or evaluated expressions to the system console. Supports string concatenation.

ParameterTypeDescription
stringText/VarThe content to display. Use @var to inject variable data.
calc(equation)

Invokes the internal math engine to solve arithmetic strings. Follows standard PEMDAS order of operations.

ParameterTypeDescription
equationExprMathematical string (e.g., "10 * (5 + 2)").
wait(seconds)

Suspends thread execution for a specified duration.

ParameterTypeDescription
secondsInt/FloatTime to pause in seconds (e.g., 0.5 for half a second).

2. Control Flow

if(condition, code)

Evaluates a boolean condition. If true, executes the provided code block or single command.

ParameterTypeDescription
conditionBooleanComparison (e.g., x > 10).
codeBlockCommands to run, separated by ; or wrapped in {}.

3. Media Engine

video(file)

Spawns a VLC process to play high-definition video files. Requires VLC Media Player installed on the host system.

ParameterTypeDescription
filePathRelative or absolute path to .mp4, .mkv, or .avi.

4. Artificial Intelligence

tinyai(msg, var, instruct, info, temp, token)

The primary command for Local AI interaction via the .gguf model. Operates entirely offline.

ParameterTypeDescription
msgTextThe prompt or question sent to the LLM.
varIDThe variable name where the AI response will be stored.
instructTextSystem-level instructions (e.g., "You are a poet").
infoTextAdditional context or RAG data. Use .default to skip.
tempFloatCreativity/Randomness (0.1 to 1.0). Use .default for 0.7.
tokenIntMaximum response length. Use .default for 512.
ai(model, key, role, msg, var)

Connects to OpenAI-compatible endpoints. Requires an active internet connection.

ParameterTypeDescription
modelStringModel ID (e.g., "gpt-4o").
keyStringYour secret API key.
roleStringThe system role definition.
msgTextThe user prompt.
varIDVariable to store the resulting JSON/text.
